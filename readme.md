# Matrix Calculator - IF 2123
Tugas Besar 1 IF 2123 Aljabar Linier dan Geometri

## How to Run?
### MANUAL:
1. Go to src : `cd src`
2. Compile in bin : `javac -d ../bin ./codes/*.java` 
3. `cd..`
4. Go to bin : `cd bin`
5. Run in bin : `java codes.mainProgram`

Note : All class files will be inside the bin folder.
